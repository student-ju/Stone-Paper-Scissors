# ✊✋✌️ Stone Paper Scissors - Interactive Web Game

A fully interactive, logic-based game built to demonstrate real-time DOM manipulation and algorithmic decision-making in JavaScript.

## 🚀 Live Demo
[Play the Game here](https://student-ju.github.io/Stone-Paper-Scissors/)

## 🧠 Engineering Logic
- **Randomized AI Opponent:** Implemented an automated computer move generator using `Math.random()` and array indexing to simulate a fair opponent.
- **State Management:** Created a centralized scoring system that tracks User vs. Computer wins dynamically throughout the session.
- **Conditional Win/Loss Matrix:** Developed an efficient logic-check algorithm to determine the winner based on the classic "Rock-Paper-Scissors" rules.

## 🛠️ Tech Stack
- **HTML5:** Structured the game arena and interactive choice elements.
- **CSS3:** Used Flexbox for a responsive UI and custom hover effects for better user engagement.
- **JavaScript (ES6):** Handled the core game engine, event listeners, and real-time UI updates.

## ✨ Features
- **Instant Feedback:** The UI updates immediately to show who won the round.
- **Score Tracking:** A persistent score counter for the duration of the play session.
- **Responsive Design:** Optimized for both desktop and mobile browsers.
